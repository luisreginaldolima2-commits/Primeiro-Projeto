## 📋 Sobre o Projeto
Este é um script simples em **Python** desenvolvido para ajudar no controle doméstico, calculando o consumo mensal de energia de aparelhos eletroeletrônicos e fornecendo uma estimativa de custo em Reais.

## 🧮 Fórmula Utilizada
Para chegar ao resultado, o sistema utiliza o seguinte cálculo:

$$consumoMensal = \frac{potencia \times horasDia \times 30}{1000}$$

## 🚀 Como executar
1. Certifique-se de ter o **Python 3** instalado.
2. Clone este repositório ou baixe o arquivo `app.py`.
3. Abra o terminal e execute o comando:
   ```bash
   python app.py
